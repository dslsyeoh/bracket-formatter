Simple Bracket Formatter
-

Overview
--
1. Formalize mathematics equations to bracket.
2. Support only basic operations listed belows
    - `Addition`
    - `Subtraction`
    - `Multiplication`
    - `Division`

Sample Input
--
1. `1 + 2 - 3 + (1 + 3) == 6`
2. `1 / 2 / 3 / 4 * 5 * 6 + 7 + 8 * 9`
3. `1 - 2 - 3 * 4 + 5 / 6 - 7 * 8 + 9 == 6`

Sample Output
--
1. `1+2-3+(1+3)`
2. `(1/2/3/4*5*6)+7+(8*9)`
3. `1-2-(3*4)+(5/6)-(7*8)+9`
